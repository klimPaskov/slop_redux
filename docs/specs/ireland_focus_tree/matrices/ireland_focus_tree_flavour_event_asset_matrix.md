# Ireland flavour event asset matrix

This matrix extends the main asset prompt for the flavour event layer. Use asset families, not a unique image for every row.

| Asset family | Event pools | Source mode | Target size | Use rule |
| --- | --- | --- | --- | --- |
| `flavour_report_civic_life` | constitutional civic life, Dail, county councils, petitions | generated period civic scene unless a real document or leader is depicted | 210x176 | report events for civic administration and public authority |
| `flavour_report_press_radio` | press, radio, censorship, public information | sourced archival press or radio material when specific, generated generic scene otherwise | 210x176 | report events and rare news variants |
| `flavour_report_language_school` | language, folklore, schools, civic culture | sourced school or Conradh material when specific, generated classroom scene otherwise | 210x176 | civic culture event family |
| `flavour_report_sport_culture` | GAA, clubs, public culture | sourced when depicting real people or organisations, generated generic sports crowd otherwise | 210x176 | civic sport and local morale events |
| `flavour_report_relief_red_cross` | Red Cross, parish relief, child relief, postwar aid | sourced Irish Red Cross material when real, generated relief scene otherwise | 210x176 | humanitarian report events |
| `flavour_report_turf_rationing` | turf, ration books, winter heat, coupons | sourced or generated period home front material | 210x176 | home front events |
| `flavour_report_rural_industry` | sugar beet, creameries, mills, workshops | sourced factory or rural industry material when specific, generated local industry scene otherwise | 210x176 | industry flavour events |
| `flavour_report_power_transport` | Ardnacrusha, ESB, rail, bus fuel, aviation | sourced where real infrastructure is shown, generated generic maintenance scene otherwise | 210x176 | power and transport events |
| `flavour_report_ports_coast` | Treaty Ports, LOPs, lighthouses, fishermen, merchant marine | sourced for real LOP or port photos, generated non specific coast scene otherwise | 210x176 | coast and shipping events |
| `flavour_report_civil_defence` | ARP, blackout, fire service, shelters, LDF drills | sourced Irish material where available, generated period civil defence otherwise | 210x176 | civil defence events |
| `flavour_report_g2_internment` | G2, internment, crash salvage, wireless checks | sourced when depicting real sites or people, generated document room scene otherwise | 210x176 | intelligence events |
| `flavour_report_border_life` | border markets, customs, rail crossings, local safeguards | generated local border scene unless a real place is depicted | 210x176 | Northern local events |
| `flavour_icon_coupon_supply` | rationing, price boards, coupon audits | generated icon art | 32x32 | decision icon family |
| `flavour_icon_civic_school` | language, schools, folklore, library | generated icon art | 32x32 or 64x64 by use | decision and idea icon family |
| `flavour_icon_coast_watch` | LOP, lighthouse, air incidents | generated icon art unless using sourced symbol | 32x32 | decision icon family |
| `flavour_icon_public_order` | marches, Garda orders, public meetings | generated icon art | 32x32 | decision icon family |
| `flavour_icon_relief` | Red Cross, parish relief, child welfare | generated icon art or sourced symbol when required | 32x32 | decision icon family |
