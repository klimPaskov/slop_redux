## Minimum acceptance columns for audits

| Audit question | Required proof |
| --- | --- |
| Is the route hidden without being undocumented | reveal flags, tooltip hints, and spec mapped conditions exist |
| Does it change play | it unlocks decisions, missions, mechanic changes, AI changes, assets, or route payoffs |
| Does it avoid fairy dust | rewards are not tiny standalone modifiers or repeated free units |
| Does it clean up | route end, invalid target, war end, tag switch, and formation cleanup are defined |
| Does AI respect validity | route AI has blockers for missing states, invalid sponsors, closed routes, and impossible partners |
| Does localisation stay safe | working labels are not final text and source dependent wording remains gated |
