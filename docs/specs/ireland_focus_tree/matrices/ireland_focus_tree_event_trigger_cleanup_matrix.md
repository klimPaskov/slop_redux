# Ireland focus tree event trigger and cleanup matrix

| Event group | Trigger owner | Repeat rule | State or actor target | Cleanup requirement |
| --- | --- | --- | --- | --- |
| opening trunk events | opening focuses and first route convention | mostly fire once | Ireland | mark route education complete and clear branch preview flags after route lock |
| port and coast events | ports focus, coast watch missions, invasion risk | fire once per milestone, crisis events can repeat through cooldown | Irish port and coast state groups | cancel when ports are lost, transferred, or war state invalidates the mission |
| public order events | BOP bands, authority missions, radical pressure | crisis events can recur with cooldown and escalating consequences | Ireland and named state groups | clear when route settles, actor disbands, or crackdown resolves |
| foreign access events | sponsor decisions and pressure thresholds | sponsor chain can repeat by sponsor with cooldown | Britain, Germany, Italy, Spain, United States, Soviet Union, Vatican channel | clear when sponsor is invalid, at incompatible war state, or dependency route closes |
| Northern events | settlement focuses, missions, British response, unionist alarm | stage events fire once, backlash can recur | Northern state groups, Britain, unionist actor abstraction | clear after settlement, defeat, protectorate end, or integration completion |
| hidden path events | reveal flags, BOP extremes, special route thresholds | reveal fires once, crisis follow ups can branch | Ireland and relevant route actor | clear if hidden route is rejected, failed, recovered, or disqualified |
| BOP events | BOP mode switch and range entry | range entry fires once per mode unless leaving and re entering after cooldown | Ireland | clear stale range flags when mode changes or country invalidates |
| formation events | formation decision and integration milestones | public formation fires once per identity | Ireland and Northern state groups | remove obsolete formation decisions and pre formation crisis flags |
