## Correction addressed

The package now treats flavour events as required playable content. The major event suite covers route milestones and crisis events. The flavour layer covers the ordinary civic, wartime, social, cultural, and local pressures that make the Ireland campaign feel alive.

## Design result

The flavour layer adds 169 required working handles or merged dynamic equivalents. It covers rationing, fuel, turf, wheat, shipping, coastwatching, censorship, radio, schools, Irish language policy, folklore, GAA travel restrictions, church and relief committees, LDF drills, rural markets, sugar beet, electricity, Labour workplaces, border families, diaspora ties, route micro flavour, and postwar memory.

## Implementation acceptance

The final implementation report must include a flavour event coverage table. Empty flavour popups do not satisfy the package. Each final flavour event must move BOP, a mechanic value, local support, decision access, mission state, state modifier, idea lifecycle, AI strategy, foreign pressure, achievement memory, or cleanup.
