# Ireland BOP bibliography

Sources listed here support design anchors. They do not provide final localisation.

## HOI4 modding reference

- Paradox Wiki, Balance of power modding. Use the offline repository snapshot during implementation when available. Web search result also identifies that balance of power definitions belong to the HOI4 BOP system and that a BOP can contain more than two sides while only two sides are visible to the player at a time.
- Paradox Wiki, Effects. Search result notes BOP definitions use `/Hearts of Iron IV/common/bop/*.txt` and effects can set or modify balance of power.
