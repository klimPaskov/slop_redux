
# Ireland focus tree specification part 4: detailed focus branches

This file continues from `ireland_focus_tree_spec_part_3_historical_route_notes.md` after `Alternate history boundaries already established`.

All focus labels in this file are working labels, not final localisation. They are internal handles for planning, implementation prompts, route matrices, asset handoffs, and audit coverage. Final focus titles, descriptions, decision names, event text, achievement titles, slogans, quotes, and audio choices remain blocked until implementation or text and audio research produces sourced and appropriate wording.

The design target remains a large playable Ireland tree. The exact final focus count, grid coordinates, prerequisites, bypasses, and final focus ids belong to implementation. This file defines the route logic, focus groups, lock rules, rewards, linked decisions, AI behavior, failure states, and asset direction that implementation must preserve.

## Part 4 branch rules

- The tree should feel like a country identity, not a reward catalogue.
- Focuses should unlock decisions, missions, advisors, leader changes, unit systems, buildings, diplomatic actions, Northern settlement tools, mechanic values, or route locks.
- Political paths should alter parties, leaders, advisors, laws, country identity direction, foreign relations, and available support decisions.
- Support branches should remain useful for several political routes, but route locks should change costs, risks, and rewards.
- The Northern path must verify state control and settlement conditions through decisions. A focus can prepare or reveal unity, but it should not grant the whole island by itself.
- Foreign access is a pressure system. Britain, Germany, Italy, Spain, the United States, the Soviet Union, and Vatican links should not be generic relation modifiers.
- The IRA foreign contact path and Plan Kathleen style content are dangerous exposure mechanics, not clean power rewards.
- Fine Gael constitutional opposition and Blueshirt corporatism are separate route families.
- Labour includes a democratic socialist path and a radical internationalist option. It is not only a Soviet branch.
- Hidden paths are accepted in part 10. Civic cultural restoration, Emergency directorate, Atlantic neutral compact, and route overlays must be implemented through reveal gates, blockers, decisions, AI, assets, and achievements.
