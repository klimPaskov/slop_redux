# Ireland Focus Tree Flavour Report Art Manifest

Generated shared report asset families for `ireland_focus_tree` flavour events.

- Source mode: `$imagegen` built-in tool.
- Final DDS folder: `gfx/event_pictures/ireland_focus_tree/`
- Source PNGs: `docs/assets/ireland_focus_tree_flavour_events/source_png/`
- Processed PNGs: `docs/assets/ireland_focus_tree_flavour_events/processed_png/`
- Contact sheet: `docs/assets/ireland_focus_tree_flavour_events/contact_sheets/ireland_flavour_report_images_contact_sheet.png`
- Status: complete unless noted.

Generation was appropriate because these are generic shared flavour families for ordinary civic, social, economic, civil-defence, border, intelligence, and relief events. They are not intended to depict specific real people, real institutions, real documents, real places, real flags, or protected/historically attested emblems.
