# Slop Redux

**Slop Redux** is a community-driven Hearts of Iron IV mod created as part of the **Vibe Coding HOI4** video series.

The idea is simple: random features, quality of life changes, focus trees, events, decisions, and other mod content are added through vibe coding. The mod grows episode by episode, with ideas coming from the community.

This is not meant to be a serious total overhaul. It is a follow-along modding project where the goal is to see what can be built with AI-assisted development, and community suggestions.

## About the series

Slop Redux is developed in public through the **Vibe Coding HOI4** series.

Each episode focuses on adding something new to the mod. The process is shown openly, including planning, prompting, and implementation.

The series is also meant to help people learn how to vibe code HOI4 mods, or how to use AI-assisted coding in general.

## Installation

### Steam Workshop

The easiest way to install the mod is through the Steam Workshop.

Steam Workshop: `ADD WORKSHOP LINK HERE`

Subscribe to the mod, enable it in the Paradox Launcher, and launch Hearts of Iron IV.

### Manual install

Clone the repository into your Hearts of Iron IV mod folder:

```bash
git clone ADD_REPOSITORY_LINK_HERE
```

Then enable the mod in the Paradox Launcher.

## Development setup

This mod is developed with an agentic HOI4 modding workflow.

Recommended tools:

* Git
* Codex or another coding agent
* Hearts of Iron IV in debug mode
* A structured `AGENTS.md` file
* Project-specific skills and planning documents

The goal is to keep the project organized enough that AI agents can work on it without constantly rediscovering the same patterns.

## Contributing

The main way to contribute is by suggesting ideas through Discord or comments on the video series.

You can also open GitHub issues for:

* Bugs
* Broken localisation
* Broken focus effects
* Balance problems
* Missing icons
* Suggestions
* Compatibility issues

Before reporting a bug, check whether someone has already reported the same issue.

## Status

This mod is in early development.

Expect bugs, unfinished content, strange balance, broken ideas, and features that probably should not exist.

That is part of the point.
